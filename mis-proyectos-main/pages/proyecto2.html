calculadora en python

Este proyecto sobre una calculadora en Python define funciones para operaciones básicas y valida los datos ingresados por el usuario. El ciclo principal permite elegir la operación, ingresar los números y muestra el resultado o un mensaje de error si hay algún problema, como división por cero u opción incorrecta. Al final de cada operación, pregunta si se desea continuar o salir.

En esta parte del código se definen las funciones para cada operación matemática de la calculadora: suma, resta, multiplicación, división, potencia y raíz. Cada función recibe dos números y retorna el resultado de la operación correspondiente. La función de división incluye una validación para evitar la división por cero, devolviendo None si ocurre. Al final, se imprime un menú con las opciones disponibles para que el usuario elija la operación que desea realizar.

Aquí se encuentra el ciclo principal del programa. El usuario selecciona la operación que desea realizar e ingresa dos números. El código valida que los datos sean correctos y ejecuta la función correspondiente según la opción elegida. Si el usuario intenta dividir por cero, el programa muestra un mensaje de error. Al finalizar cada operación, pregunta si desea continuar o salir. Si el usuario ingresa una opción incorrecta, el programa muestra un mensaje de error y vuelve a pedir la operación.
